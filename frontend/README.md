# Frontend — MF Genie

Next.js 15 app for the Mutual Fund Genie chat UI.

- **Setup:** From repo root see [README.md](../README.md). From here: `npm install`
- **Dev:** `npm run dev` → http://localhost:3000
- **Build:** `npm run build`
- **Tests:** `npm test`

API base URL is set via `NEXT_PUBLIC_API_URL` (defaults to same origin). For local RAG backend, run Phase-4_Backend_API on port 8000 and use `http://localhost:8000` if needed.
